# Changelog

All notable changes to ScanWise will be documented in this file.

## [0.1.1] - 2026-06-28

### Added
- Windows installer (`ScanWise_Setup.exe`) built with Inno Setup.
- `tools/build_installer.ps1` local installer build script.

## [0.1.0] - 2026-06-28

### Added
- Initial release of ScanWise.
- Drag & drop and clipboard image import.
- Automatic document edge detection (Magic Perspective) with manual corner adjustment.
- Perspective warp/crop flattening.
- Filter presets: Original, Black & White, Document, Magic Color, and Photo.
- Brightness, contrast, and saturation adjustments.
- Multi-page list with add, remove, and reorder.
- Single-page PNG / JPEG export and multi-page PDF export.
- Engine unit tests for document corner detection.
